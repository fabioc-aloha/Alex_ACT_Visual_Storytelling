# delivery-html-dashboard

Self-contained HTML dashboard with Chart.js.

**Status**: Planned (Phase 2)

## Scope

- Single HTML file, zero build step, open in any browser
- Chart.js 4.x via CDN for interactive charts
- KPI card components with metric + delta + sparkline
- CSS grid responsive layout with auto-fit
- Filter architecture (global, chart-level, cross-filter)
- Embedded data (JavaScript arrays, no external API calls)
- CSS custom properties for theming
- Print-friendly @media styles
- WCAG 2.1 AA accessibility

## Pipeline Position

Final step. Alternative to `delivery-svg-markdown` when interactivity is needed.
